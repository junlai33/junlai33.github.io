# [“我没事”背后，是千言万语的沉默](https://hoo.be/ribenyiqu)
“我没事”背后，是千言万语的沉默
“你怎么了？”
“我没事。”
“真的没事吗？感觉你不太高兴。”
“真的，就是有点累。”

这段对话，是否似曾相识？在无数亲密关系中，“我没事”成了一句最标准的谎话，一句终止沟通的休止符。它的背后，可能藏着委屈、失望、需要被理解的渴望，或是害怕冲突的退缩。

现代人崇尚独立与坚强，尤其在许多女性心中，过度表达情绪被视为一种“作”和不成熟。于是，我们学会了吞咽情绪，用“没事”来维持表面的和平。而男性则常常被“解决问题”的思维主导，当伴侣倾诉时，第一反应是提供方案，而非给予共情，这反而让对方感到不被理解，从而关上心门。

启示： 爱，需要穿透“我没事”这堵墙的智慧。它要求我们不仅用耳朵听，更要用心去感受。当对方说“我没事”时，或许一个无声的拥抱、一句“我在这里陪着你”，比千万句追问更能触及真心。
